# Quick Notes App

A lightweight, modern web application for managing your thoughts, tasks, and daily reminders. Built purely with semantic HTML5, CSS3, and Vanilla JavaScript, this app stores data locally so your notes are never lost.

## 🚀 Live Preview / Core Features

* **Full CRUD Operations:** Seamlessly Add, Read, Update (Edit), and Delete notes.
* **Local Storage Persistence:** Your notes and theme preferences are saved securely inside your browser, meaning they stay intact even if you close or refresh the tab.
* **Native Dialog Modals:** Uses the HTML5 `<dialog>` element for ultra-smooth creation and editing popups.
* **Dark Mode Toggle:** Easy shift between light and dark themes with persistent preference saving.
* **Responsive Grid:** Automatically adjusts its layout cleanly across desktop, tablet, and mobile displays.

## 🛠️ Built With

* **HTML5** (Utilizing modern semantic layout and native `<dialog>`)
* **CSS3** (Grid layouts and modern CSS variable-based theme switching)
* **Vanilla JavaScript** (ES6 arrays, Map functions, and Browser LocalStorage API)
